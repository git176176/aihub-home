# AIHub 首页与广告位

为 **OneNav / 一为导航主题** 适配的 WordPress 插件，提供三块功能，全部可在后台开关，不修改主题文件。

## 功能

1. **首页工具卡片**：后台可视化管理（增删改、排序、启用），前端用短代码 `[aihub_cards]` 渲染。复刻图钉/渐变/折角卡片样式。支持从现有 `card-data.json` 一键导入。
2. **快讯跑马灯**（横向连续滚动）：内容来自 NewsFlash 快讯插件 + 广告混排。首页用短代码 `[aihub_ticker]`，文章页自动注入到面包屑（`.crumbs`）上方。
3. **侧边栏广告卡**：文章页侧边栏注入纯海报图（必填海报 URL），无边框/标题/角标。
4. **横版海报组**：窄长横版海报，一行 2~3 个（后台可调），支持多个独立海报位（短代码 `[aihub_banner slot=标识]`）。海报多于一行自动淡入淡出轮播。
5. **广告统计**：覆盖跑马灯广告 / 侧边栏海报 / 横版海报，统计展示次数 + 点击次数 + CTR。数据前端实时上报，不受页面缓存影响。

## 安装

1. 后台 → 插件 → 上传插件 → 选 `aihub-home-v1.0.0.zip` → 启用
2. 进左侧菜单「AIHub」配置
3. 首页：把原来手写卡片的 HTML 删掉，换成 `[aihub_ticker]`（上）+ `[aihub_cards]`（下）
4. 文章页跑马灯 / 侧边栏广告卡：在「跑马灯」「侧边栏」Tab 打开开关即可，自动注入

## 后台 Tab

- **卡片**：管理首页卡片 + JSON 导入
- **跑马灯**：开关 / 快讯条数 / 广告混入频率 / 滚动速度 / 首页·文章页分别开关
- **广告**：广告库（标题+链接，可选图标/描述），投放到 跑马灯/侧边栏/两者
- **侧边栏**：开关 / 卡片标题 / 合并或独立 / 插入位置
- **设置**：总开关 + 锚点选择器（主题改版时自救用）

## 兼容性

- 不改任何主题文件，纯短代码 + 前端 JS 注入
- 注入前判断锚点存在，不存在则静默跳过（不报错）
- 锚点选择器可在后台改，应对主题升级
- CSS 全 `aihub-` 前缀；侧边栏卡复用主题 class 求视觉一致
- NewsFlash 未安装时，跑马灯自动降级为只显示广告

## 依赖

- 跑马灯快讯数据依赖 [NewsFlash 快讯插件](https://github.com/git176176/newsflash)（可选，未装则只出广告）
